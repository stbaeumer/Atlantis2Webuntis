# Atlantis2Webuntis #

## ... erstellt eine CSV-Datei für den Import von Atlantis-Schülerdaten und eine Zipdatei mit den Schülerfotos in Webuntis. ##

### Vorgehen

1. Die Datei *Students.csv* muss als admin aus Webuntis exoprtiert und auf den Desktop gelegt werden. Starten Sie *Atlantis2Webuntis*, um angezeigt zu bekommen, wie die Exportdatei zu erstellen ist. 
2. Sobald die Datei *Students.csv* auf dem Desktop liegt, werden nach dem Programmstart von *Atlantis2Webuntis* nacheinander die Betriebe, Adressen und Schülerdaten aus Atlantis und die Schülerdaten aus der Datei *Students.csv* eingelesen.
3. Nach dem Einlesen werden alle Schülerinnen und Schüler angezeigt, die in Untis zu löschen sind. Es wird bei diesen Schülerinnen und Schülern ein Austrittsdatum gesetzt.
4. Die Importdatei für Webuntis wird geschrieben.
5. Die Fotos der neuen Schülerinnen und Schüler werden in eine Zip-Datei gepackt.
